## 属性

---
| 属性                | 类型     | 必填  | 默认值                | 说明                                    |
|:-------------------:|:--------:|:-----:|:---------------------:|:---------------------------------------:|
| modelValue          | object   | 否    | {}                    | 组件的双向绑定值。                      |
| i18n                | function | 否    | (key) => key          | 参见组件的 `i18n` 选项。                |
| foreground          | string   | 否    | 'rgba(0, 90, 158, 1)' | 参见组件的 `foreground` 选项。          |
| deleteColumnFunc    | function | 否    | null                  | 参见组件的 `deleteColumnFunc` 选项。    |
| deleteRowsFunc      | function | 否    | null                  | 参见组件的 `deleteRowsFunc` 选项。      |
| copyTableFunc       | function | 否    | null                  | 参见组件的 `copyTableFunc` 选项。       |
| pastedTableFunc     | function | 否    | null                  | 参见组件的 `pastedTableFunc` 选项。     |
| syncRowsFunc        | function | 否    | null                  | 参见组件的 `syncRowsFunc` 选项。        |
| rightMenuBackground | string   | 否    | ''                    | 参见组件的 `rightMenuBackground` 选项。 |
| extensions          | array    | 否    | []                    | 参见组件的 `extensions` 选项。          |
| theme               | string   | 否    | 'global'              | 参见组件的 `theme` 选项。               |
| disabled            | boolean  | 否    | false                 | 参见组件的 `disabled` 选项。            |
| lang                | string   | 否    | "global"              | 参见组件的 `lang` 选项。                |
