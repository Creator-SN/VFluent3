## 属性

---
| 属性             | 类型    | 必填  | 默认值       | 说明                                 |
|:----------------:|:-------:|:-----:|:------------:|:------------------------------------:|
| modelValue       | array   | 否    | []           | 组件的双向绑定值。                   |
| direction        | string  | 否    | 'horizontal' | 参见组件的 `direction` 选项。        |
| animation        | string  | 否    | 'move'       | 参见组件的 `animation` 选项。        |
| duration         | number  | 否    | 800          | 参见组件的 `duration` 选项。         |
| autoPlay         | boolean | 否    | true         | 参见组件的 `autoPlay` 选项。         |
| autoPlayDuration | number  | 否    | 5000         | 参见组件的 `autoPlayDuration` 选项。 |
| showControlPanel | string  | 否    | ''           | 参见组件的 `showControlPanel` 选项。 |
| mask             | string  | 否    | ''           | 参见组件的 `mask` 选项。             |
| theme            | string  | 否    | 'global'     | 参见组件的 `theme` 选项。            |
| disabled         | boolean | 否    | false        | 参见组件的 `disabled` 选项。         |
| lang             | string  | 否    | "global"     | 参见组件的 `lang` 选项。             |
