## 属性

---
| 属性                | 类型    | 必填  | 默认值   | 说明                                    |
|:-------------------:|:-------:|:-----:|:--------:|:---------------------------------------:|
| modelValue          | boolean | 否    | false    | 组件的双向绑定值。                      |
| rightMenuWidth      | number  | 否    | 200      | 参见组件的 `rightMenuWidth` 选项。      |
| background          | string  | 否    | ''       | 参见组件的 `background` 选项。          |
| fullExpandAnimation | boolean | 否    | false    | 参见组件的 `fullExpandAnimation` 选项。 |
| theme               | string  | 否    | 'global' | 参见组件的 `theme` 选项。               |
| disabled            | boolean | 否    | false    | 参见组件的 `disabled` 选项。            |
| lang                | string  | 否    | "global" | 参见组件的 `lang` 选项。                |
