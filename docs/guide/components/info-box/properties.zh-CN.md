## 属性

---
| 属性         | 类型    | 必填  | 默认值     | 说明                             |
|:------------:|:-------:|:-----:|:----------:|:--------------------------------:|
| title        | string  | 否    | 'Tip'      | 参见组件的 `title` 选项。        |
| status       | string  | 否    | 'default'  | 参见组件的 `status` 选项。       |
| mode         | string  | 否    | 'relative' | 参见组件的 `mode` 选项。         |
| confirmTitle | string  | 否    | '确定'     | 参见组件的 `confirmTitle` 选项。 |
| cancelTitle  | string  | 否    | '取消'     | 参见组件的 `cancelTitle` 选项。  |
| showTitleBar | boolean | 否    | true       | 参见组件的 `showTitleBar` 选项。 |
| acrylic      | boolean | 否    | false      | 参见组件的 `acrylic` 选项。      |
| destroy      |         | 否    | false      | 参见组件的 `destroy` 选项。      |
| theme        | string  | 否    | 'global'   | 参见组件的 `theme` 选项。        |
| disabled     | boolean | 否    | false      | 参见组件的 `disabled` 选项。     |
| lang         | string  | 否    | "global"   | 参见组件的 `lang` 选项。         |
