## 属性

---
| 属性             | 类型    | 必填  | 默认值    | 说明                                 |
|:----------------:|:-------:|:-----:|:---------:|:------------------------------------:|
| modelValue       | string  | 否    | ''        | 组件的双向绑定值。                   |
| length           | number  | 否    | 4         | 参见组件的 `length` 选项。           |
| inputmode        | string  | 否    | 'numeric' | 参见组件的 `inputmode` 选项。        |
| underline        | boolean | 否    | false     | 参见组件的 `underline` 选项。        |
| background       | string  | 否    | ''        | 参见组件的 `background` 选项。       |
| borderWidth      | number  | 否    | 1         | 参见组件的 `borderWidth` 选项。      |
| borderColor      | string  | 否    | ''        | 参见组件的 `borderColor` 选项。      |
| focusBorderColor | string  | 否    | ''        | 参见组件的 `focusBorderColor` 选项。 |
| fontSize         | number  | 否    | 18        | 参见组件的 `fontSize` 选项。         |
| fontWeight       | string  | 否    | 'bold'    | 参见组件的 `fontWeight` 选项。       |
| foreground       | string  | 否    | ''        | 参见组件的 `foreground` 选项。       |
| borderRadius     | number  | 否    | 3         | 参见组件的 `borderRadius` 选项。     |
| isBoxShadow      | boolean | 否    | false     | 参见组件的 `isBoxShadow` 选项。      |
| revealBorder     | boolean | 否    | false     | 参见组件的 `revealBorder` 选项。     |
| disabled         | boolean | 否    | false     | 参见组件的 `disabled` 选项。         |
| theme            | string  | 否    | 'global'  | 参见组件的 `theme` 选项。            |
| lang             | string  | 否    | "global"  | 参见组件的 `lang` 选项。             |
