## 属性

---
| 属性          | 类型    | 必填  | 默认值             | 说明                              |
|:-------------:|:-------:|:-----:|:------------------:|:---------------------------------:|
| modelValue    | number  | 否    | 0                  | 组件的双向绑定值。                |
| maxRate       | number  | 否    | 5                  | 参见组件的 `maxRate` 选项。       |
| halfRate      | boolean | 否    | false              | 参见组件的 `halfRate` 选项。      |
| defaultIcon   | string  | 否    | 'FavoriteStar'     | 参见组件的 `defaultIcon` 选项。   |
| selectedIcon  | string  | 否    | 'FavoriteStarFill' | 参见组件的 `selectedIcon` 选项。  |
| borderColor   | string  | 否    | ''                 | 参见组件的 `borderColor` 选项。   |
| selectedColor | string  | 否    | ''                 | 参见组件的 `selectedColor` 选项。 |
| isClear       | boolean | 否    | false              | 参见组件的 `isClear` 选项。       |
| readonly      | boolean | 否    | false              | 参见组件的 `readonly` 选项。      |
| theme         | string  | 否    | 'global'           | 参见组件的 `theme` 选项。         |
| disabled      | boolean | 否    | false              | 参见组件的 `disabled` 选项。      |
| lang          | string  | 否    | "global"           | 参见组件的 `lang` 选项。          |
