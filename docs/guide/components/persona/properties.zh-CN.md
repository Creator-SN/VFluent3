## 属性

---
| 属性       | 类型    | 必填  | 默认值   | 说明                           |
|:----------:|:-------:|:-----:|:--------:|:------------------------------:|
| src        | string  | 否    | ''       | 参见组件的 `src` 选项。        |
| name       | string  | 否    | ''       | 参见组件的 `name` 选项。       |
| size       | number  | 否    | 40       | 参见组件的 `size` 选项。       |
| status     | string  | 否    | ''       | 参见组件的 `status` 选项。     |
| icon       | string  | 否    | ''       | 参见组件的 `icon` 选项。       |
| iconColor  | string  | 否    | ''       | 参见组件的 `iconColor` 选项。  |
| background | string  | 否    | ''       | 参见组件的 `background` 选项。 |
| showInfo   | boolean | 否    | false    | 参见组件的 `showInfo` 选项。   |
| theme      | string  | 否    | 'global' | 参见组件的 `theme` 选项。      |
| disabled   | boolean | 否    | false    | 参见组件的 `disabled` 选项。   |
| lang       | string  | 否    | "global" | 参见组件的 `lang` 选项。       |
