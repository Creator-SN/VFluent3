## 属性

---
| 属性                    | 类型    | 必填  | 默认值        | 说明                                        |
|:-----------------------:|:-------:|:-----:|:-------------:|:-------------------------------------------:|
| modelValue              | object  | 否    | {}            | 组件的双向绑定值。                          |
| options                 | array   | 否    | []            | 参见组件的 `options` 选项。                 |
| borderWidth             | number  | 否    | 1             | 参见组件的 `borderWidth` 选项。             |
| placeholder             | string  | 否    | 'Combobox'    | 参见组件的 `placeholder` 选项。             |
| borderColor             | string  | 否    | ''            | 参见组件的 `borderColor` 选项。             |
| borderRadius            | number  | 否    | 6             | 参见组件的 `borderRadius` 选项。            |
| background              | string  | 否    | ''            | 参见组件的 `background` 选项。              |
| choosenBackground       | string  | 否    | ''            | 参见组件的 `choosenBackground` 选项。       |
| choosenSliderBackground | string  | 否    | ''            | 参见组件的 `choosenSliderBackground` 选项。 |
| inputForeground         | string  | 否    | ''            | 参见组件的 `inputForeground` 选项。         |
| inputBackground         | string  | 否    | ''            | 参见组件的 `inputBackground` 选项。         |
| titleForeground         | string  | 否    | ''            | 参见组件的 `titleForeground` 选项。         |
| itemHeight              | number  | 否    | 40            | 参见组件的 `itemHeight` 选项。              |
| dropDownIcon            | string  | 否    | 'ChevronDown' | 参见组件的 `dropDownIcon` 选项。            |
| dropDownIconForeground  | string  | 否    | ''            | 参见组件的 `dropDownIconForeground` 选项。  |
| revealBorderColor       | boolean | 否    | false         | 参见组件的 `revealBorderColor` 选项。       |
| revealBackgroundColor   | boolean | 否    | false         | 参见组件的 `revealBackgroundColor` 选项。   |
| disabled                | boolean | 否    | false         | 参见组件的 `disabled` 选项。                |
| theme                   | string  | 否    | 'global'      | 参见组件的 `theme` 选项。                   |
| lang                    | string  | 否    | "global"      | 参见组件的 `lang` 选项。                    |
