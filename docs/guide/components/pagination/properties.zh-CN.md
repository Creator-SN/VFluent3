## 属性

---
| 属性                  | 类型    | 必填  | 默认值                                                          | 说明                                      |
|:---------------------:|:-------:|:-----:|:---------------------------------------------------------------:|:-----------------------------------------:|
| modelValue            | number  | 否    | 1                                                               | 组件的双向绑定值。                        |
| emits                 | array   | 否    | ['update:modelValue', 'next-click', 'prev-click', 'page-click'] | 参见组件的 `emits` 选项。                 |
| foreground            | string  | 否    | 'rgba(0, 90, 158, 1)'                                           | 参见组件的 `foreground` 选项。            |
| background            | string  | 否    | 'transparent'                                                   | 参见组件的 `background` 选项。            |
| borderRadius          | number  | 否    | 3                                                               | 参见组件的 `borderRadius` 选项。          |
| revealBackgroundColor | string  | 否    | ''                                                              | 参见组件的 `revealBackgroundColor` 选项。 |
| revealBorderColor     | string  | 否    | ''                                                              | 参见组件的 `revealBorderColor` 选项。     |
| total                 | number  | 否    | 10                                                              | 参见组件的 `total` 选项。                 |
| maxVisual             | number  | 否    | 7                                                               | 参见组件的 `maxVisual` 选项。             |
| small                 | boolean | 否    | false                                                           | 参见组件的 `small` 选项。                 |
| shadow                | boolean | 否    | false                                                           | 参见组件的 `shadow` 选项。                |
| disabled              | boolean | 否    | false                                                           | 参见组件的 `disabled` 选项。              |
| theme                 | string  | 否    | 'global'                                                        | 参见组件的 `theme` 选项。                 |
| lang                  | string  | 否    | "global"                                                        | 参见组件的 `lang` 选项。                  |
