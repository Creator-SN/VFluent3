## 属性

---
| 属性                   | 类型    | 必填  | 默认值        | 说明                                       |
|:----------------------:|:-------:|:-----:|:-------------:|:------------------------------------------:|
| modelValue             | array   | 否    | []            | 组件的双向绑定值。                         |
| borderWidth            | number  | 否    | 1             | 参见组件的 `borderWidth` 选项。            |
| placeholder            | string  | 否    | 'Pick a day'  | 参见组件的 `placeholder` 选项。            |
| dropDownBorderRadius   | number  | 否    | 6             | 参见组件的 `dropDownBorderRadius` 选项。   |
| inputForeground        | string  | 否    | ''            | 参见组件的 `inputForeground` 选项。        |
| inputBackground        | string  | 否    | ''            | 参见组件的 `inputBackground` 选项。        |
| dropDownIcon           | string  | 否    | 'CalendarDay' | 参见组件的 `dropDownIcon` 选项。           |
| dropDownIconForeground | string  | 否    | ''            | 参见组件的 `dropDownIconForeground` 选项。 |
| dropDownRevealBorder   | boolean | 否    | true          | 参见组件的 `dropDownRevealBorder` 选项。   |
| dropDownIsBoxShadow    | boolean | 否    | true          | 参见组件的 `dropDownIsBoxShadow` 选项。    |
| dropDownStyles         | object  | 否    | {}            | 参见组件的 `dropDownStyles` 选项。         |
| editable               | boolean | 否    | false         | 参见组件的 `editable` 选项。               |
| multiple               | string  | 否    | 'single'      | 参见组件的 `multiple` 选项。               |
| choosenDates           | array   | 否    | []            | 参见组件的 `choosenDates` 选项。           |
| background             | string  | 否    | ''            | 参见组件的 `background` 选项。             |
| borderRadius           | string  | 否    | ''            | 参见组件的 `borderRadius` 选项。           |
| foreground             | string  | 否    | ''            | 参见组件的 `foreground` 选项。             |
| nowDayColor            | string  | 否    | ''            | 参见组件的 `nowDayColor` 选项。            |
| rangeChooseColorFE     | string  | 否    | ''            | 参见组件的 `rangeChooseColorFE` 选项。     |
| rangeChooseColorMiddle | string  | 否    | ''            | 参见组件的 `rangeChooseColorMiddle` 选项。 |
| disabled               | boolean | 否    | false         | 参见组件的 `disabled` 选项。               |
| theme                  | string  | 否    | 'global'      | 参见组件的 `theme` 选项。                  |
| lang                   | string  | 否    | "global"      | 参见组件的 `lang` 选项。                   |
