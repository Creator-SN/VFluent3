## 属性

---
| 属性               | 类型    | 必填  | 默认值   | 说明                                   |
|:------------------:|:-------:|:-----:|:--------:|:--------------------------------------:|
| modelValue         | boolean | 否    | false    | 组件的双向绑定值。                     |
| on                 | string  | 否    | 'On'     | 参见组件的 `on` 选项。                 |
| off                | string  | 否    | 'Off'    | 参见组件的 `off` 选项。                |
| onForeground       | string  | 否    | ''       | 参见组件的 `onForeground` 选项。       |
| offForeground      | string  | 否    | ''       | 参见组件的 `offForeground` 选项。      |
| width              | number  | 否    | 40       | 参见组件的 `width` 选项。              |
| height             | number  | 否    | 20       | 参见组件的 `height` 选项。             |
| borderColor        | string  | 否    | ''       | 参见组件的 `borderColor` 选项。        |
| ringBackground     | string  | 否    | ''       | 参见组件的 `ringBackground` 选项。     |
| switchOnBackground | string  | 否    | ''       | 参见组件的 `switchOnBackground` 选项。 |
| insideContent      | boolean | 否    | false    | 参见组件的 `insideContent` 选项。      |
| disabled           | boolean | 否    | false    | 参见组件的 `disabled` 选项。           |
| theme              | string  | 否    | 'global' | 参见组件的 `theme` 选项。              |
| lang               | string  | 否    | "global" | 参见组件的 `lang` 选项。               |
