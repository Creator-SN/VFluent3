## 属性

---
| 属性     | 类型    | 必填  | 默认值   | 说明                         |
|:--------:|:-------:|:-----:|:--------:|:----------------------------:|
| left     | string  | 否    | ''       | 参见组件的 `left` 选项。     |
| top      | string  | 否    | ''       | 参见组件的 `top` 选项。      |
| right    | string  | 否    | ''       | 参见组件的 `right` 选项。    |
| bottom   | string  | 否    | ''       | 参见组件的 `bottom` 选项。   |
| target   | any     | 否    | null     | 参见组件的 `target` 选项。   |
| mode     | string  | 否    | 'fixed'  | 参见组件的 `mode` 选项。     |
| affixPos | array   | 否    | ['top']  | 参见组件的 `affixPos` 选项。 |
| zIndex   | string  | 否    | ''       | 参见组件的 `zIndex` 选项。   |
| theme    | string  | 否    | 'global' | 参见组件的 `theme` 选项。    |
| disabled | boolean | 否    | false    | 参见组件的 `disabled` 选项。 |
| lang     | string  | 否    | "global" | 参见组件的 `lang` 选项。     |
