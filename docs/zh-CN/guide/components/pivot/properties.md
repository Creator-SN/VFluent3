## 属性

---
| 属性               | 类型    | 必填  | 默认值                         | 说明                                   |
|:------------------:|:-------:|:-----:|:------------------------------:|:--------------------------------------:|
| modelValue         | object  | 否    | () => { return null; }         | 组件的双向绑定值。                     |
| items              | array   | 否    | [{ name: 'Pivot', width: 80 }] | 参见组件的 `items` 选项。              |
| tab                | boolean | 否    | false                          | 参见组件的 `tab` 选项。                |
| padding            | string  | 否    | ''                             | 参见组件的 `padding` 选项。            |
| itemPadding        | string  | 否    | ''                             | 参见组件的 `itemPadding` 选项。        |
| fontSize           | string  | 否    | ''                             | 参见组件的 `fontSize` 选项。           |
| foreground         | string  | 否    | ''                             | 参见组件的 `foreground` 选项。         |
| choosenForeground  | string  | 否    | ''                             | 参见组件的 `choosenForeground` 选项。  |
| sliderBackground   | string  | 否    | ''                             | 参见组件的 `sliderBackground` 选项。   |
| sliderBoxshadow    | boolean | 否    | false                          | 参见组件的 `sliderBoxshadow` 选项。    |
| sliderBorderRadius | string  | 否    | '3'                            | 参见组件的 `sliderBorderRadius` 选项。 |
| background         | string  | 否    | ''                             | 参见组件的 `background` 选项。         |
| borderRadius       | string  | 否    | ''                             | 参见组件的 `borderRadius` 选项。       |
| theme              | string  | 否    | 'global'                       | 参见组件的 `theme` 选项。              |
| disabled           | boolean | 否    | false                          | 参见组件的 `disabled` 选项。           |
| lang               | string  | 否    | "global"                       | 参见组件的 `lang` 选项。               |
