## 属性

---         
| 属性             | 类型      | 必填  | 默认值    | 说明                             |
|:----------------:|:---------:|:-----:|:---------:|:--------------------------------:|
| background       | string    | 否    | ''        | background color                 |
| disabled         | boolean   | 否    | false     | disabled                         |
| error            | [boolean] | No    | false     | error state                      |
| error-foreground | [string]  | No    | undefined | error foreground color           |
| foreground       | string    | 否    | ''        | foreground color                 |
| indeterminate    | [boolean] | No    | false     | indeterminate status             |
| lang             | string    | 否    | "global"  | i18n                             |
| max              | [number]  | No    | 100       | same as HTML progress element    |
| modelValue       | number    | 否    | 0         | v-model                          |
| pause            | [boolean] | No    | false     | pause state                      |
| pause-foreground | [string]  | No    | undefined | pause foreground color           |
| theme            | string    | 否    | 'global'  | theme                            |
| borderRadius     | number    | 否    | 2         | 参见组件的 `borderRadius` 选项。 |
| loading          | boolean   | 否    | false     | 参见组件的 `loading` 选项。      |
