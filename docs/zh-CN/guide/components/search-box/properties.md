## 属性

---
| 属性              | 类型    | 必填  | 默认值   | 说明                                  |
|:-----------------:|:-------:|:-----:|:--------:|:-------------------------------------:|
| modelValue        | string  | 否    | ''       | 组件的双向绑定值。                    |
| options           | array   | 否    | []       | 参见组件的 `options` 选项。           |
| placeholder       | string  | 否    | ''       | 参见组件的 `placeholder` 选项。       |
| type              | string  | 否    | 'text'   | 参见组件的 `type` 选项。              |
| readonly          | boolean | 否    | false    | 参见组件的 `readonly` 选项。          |
| maxlength         | string  | 否    | ''       | 参见组件的 `maxlength` 选项。         |
| customFilter      | boolean | 否    | false    | 参见组件的 `customFilter` 选项。      |
| resultPlaceholder | boolean | 否    | false    | 参见组件的 `resultPlaceholder` 选项。 |
| focusShow         | boolean | 否    | false    | 参见组件的 `focusShow` 选项。         |
| leftIcon          | string  | 否    | ''       | 参见组件的 `leftIcon` 选项。          |
| icon              | string  | 否    | ''       | 参见组件的 `icon` 选项。              |
| underline         | boolean | 否    | false    | 参见组件的 `underline` 选项。         |
| background        | string  | 否    | ''       | 参见组件的 `background` 选项。        |
| borderWidth       | number  | 否    | 1        | 参见组件的 `borderWidth` 选项。       |
| borderColor       | string  | 否    | ''       | 参见组件的 `borderColor` 选项。       |
| focusBorderColor  | string  | 否    | ''       | 参见组件的 `focusBorderColor` 选项。  |
| foreground        | string  | 否    | ''       | 参见组件的 `foreground` 选项。        |
| fontSize          | number  | 否    | 13.3     | 参见组件的 `fontSize` 选项。          |
| fontWeight        | string  | 否    | 'normal' | 参见组件的 `fontWeight` 选项。        |
| textAlign         | string  | 否    | 'left'   | 参见组件的 `textAlign` 选项。         |
| borderRadius      | number  | 否    | 3        | 参见组件的 `borderRadius` 选项。      |
| isBoxShadow       | boolean | 否    | false    | 参见组件的 `isBoxShadow` 选项。       |
| revealBorder      | boolean | 否    | false    | 参见组件的 `revealBorder` 选项。      |
| status            | string  | 否    | ''       | 参见组件的 `status` 选项。            |
| debounceDelay     | number  | 否    | 300      | 参见组件的 `debounceDelay` 选项。     |
| disabled          | boolean | 否    | false    | 参见组件的 `disabled` 选项。          |
| theme             | string  | 否    | 'global' | 参见组件的 `theme` 选项。             |
| lang              | string  | 否    | "global" | 参见组件的 `lang` 选项。              |
