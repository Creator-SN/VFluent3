## 属性

---
| 属性         | 类型    | 必填  | 默认值                | 说明                             |
|:------------:|:-------:|:-----:|:---------------------:|:--------------------------------:|
| src          | string  | 否    | ''                    | 参见组件的 `src` 选项。          |
| onlazy       | boolean | 否    | false                 | 参见组件的 `onlazy` 选项。       |
| loadingColor | string  | 否    | 'rgba(36, 36, 36, 1)' | 参见组件的 `loadingColor` 选项。 |
| onbackground | boolean | 否    | false                 | 参见组件的 `onbackground` 选项。 |
| theme        | string  | 否    | 'global'              | 参见组件的 `theme` 选项。        |
| disabled     | boolean | 否    | false                 | 参见组件的 `disabled` 选项。     |
| lang         | string  | 否    | "global"              | 参见组件的 `lang` 选项。         |
