## 属性

---
| 属性       | 类型    | 必填  | 默认值    | 说明                         |
|:----------:|:-------:|:-----:|:---------:|:----------------------------:|
| modelValue | any     | 是    | undefined | 组件的双向绑定值。           |
| disabled   | boolean | 否    | false     | 参见组件的 `disabled` 选项。 |
| label      | string  | 否    | ''        | 参见组件的 `label` 选项。    |
| inline     | boolean | 否    | false     | 参见组件的 `inline` 选项。   |
| theme      | string  | 否    | 'global'  | 参见组件的 `theme` 选项。    |
| lang       | string  | 否    | "global"  | 参见组件的 `lang` 选项。     |
