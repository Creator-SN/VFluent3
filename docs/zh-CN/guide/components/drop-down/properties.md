## 属性

---
| 属性                   | 类型    | 必填  | 默认值                       | 说明                                       |
|:----------------------:|:-------:|:-----:|:----------------------------:|:------------------------------------------:|
| modelValue             | array   | 否    | []                           | 组件的双向绑定值。                         |
| options                | array   | 否    | []                           | 参见组件的 `options` 选项。                |
| multiple               | boolean | 否    | false                        | 参见组件的 `multiple` 选项。               |
| borderWidth            | number  | 否    | 1                            | 参见组件的 `borderWidth` 选项。            |
| borderRadius           | string  | 否    | '6'                          | 参见组件的 `borderRadius` 选项。           |
| placeholder            | string  | 否    | 'Dropdown'                   | 参见组件的 `placeholder` 选项。            |
| maxHeight              | string  | 否    | ''                           | 参见组件的 `maxHeight` 选项。              |
| inputForeground        | string  | 否    | ''                           | 参见组件的 `inputForeground` 选项。        |
| inputBorderColor       | string  | 否    | ''                           | 参见组件的 `inputBorderColor` 选项。       |
| inputHeight            | string  | 否    | ''                           | 参见组件的 `inputHeight` 选项。            |
| inputFontSize          | string  | 否    | ''                           | 参见组件的 `inputFontSize` 选项。          |
| checkBoxBackground     | string  | 否    | ''                           | 参见组件的 `checkBoxBackground` 选项。     |
| dropDownListForeground | string  | 否    | ''                           | 参见组件的 `dropDownListForeground` 选项。 |
| dropDownListBackground | string  | 否    | ''                           | 参见组件的 `dropDownListBackground` 选项。 |
| inputBackground        | string  | 否    | ''                           | 参见组件的 `inputBackground` 选项。        |
| dropDownIcon           | string  | 否    | 'ChevronDown'                | 参见组件的 `dropDownIcon` 选项。           |
| dropDownIconForeground | string  | 否    | ''                           | 参见组件的 `dropDownIconForeground` 选项。 |
| showError              | boolean | 否    | false                        | 参见组件的 `showError` 选项。              |
| errorMessage           | string  | 否    | 'This dropdown has an error' | 参见组件的 `errorMessage` 选项。           |
| disabled               | boolean | 否    | false                        | 参见组件的 `disabled` 选项。               |
| setFocus               | boolean | 否    | false                        | 参见组件的 `setFocus` 选项。               |
| theme                  | string  | 否    | 'global'                     | 参见组件的 `theme` 选项。                  |
| lang                   | string  | 否    | "global"                     | 参见组件的 `lang` 选项。                   |
