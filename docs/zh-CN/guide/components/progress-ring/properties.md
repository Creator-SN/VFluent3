## 属性

---         
| 属性             | 类型      | 必填  | 默认值                   | 说明                           |
|:----------------:|:---------:|:-----:|:------------------------:|:------------------------------:|
| borderWidth      | number    | 否    | 8                        | circle border width            |
| disabled         | boolean   | 否    | false                    | disabled                       |
| error            | [boolean] | No    | false                    | error state                    |
| error-foreground | [string]  | No    | undefined                | error state circle color       |
| foreground       | [string]  | No    | undefined                | circle color                   |
| indeterminate    | [boolean] | No    | false                    | indeterminate state            |
| lang             | string    | 否    | "global"                 | i18n                           |
| max              | [number]  | No    | 100                      | same as progress max property  |
| modelValue       | number    | 否    | 0                        | v-model                        |
| pause            | [boolean] | No    | false                    | pause state                    |
| pause-foreground | [string]  | No    | undefined                | pause state circle color       |
| theme            | string    | 否    | 'global'                 | theme                          |
| width            | [number]  | No    | 60                       | circle width                   |
| size             | string    | 否    | 'xs'                     | 参见组件的 `size` 选项。       |
| r                | number    | 否    | 40                       | 参见组件的 `r` 选项。          |
| loading          | boolean   | 否    | false                    | 参见组件的 `loading` 选项。    |
| color            | string    | 否    | 'rgba(0, 90, 158, 1)'    | 参见组件的 `color` 选项。      |
| background       | string    | 否    | 'rgba(204, 204, 204, 1)' | 参见组件的 `background` 选项。 |
| legacy           | boolean   | 否    | false                    | 参见组件的 `legacy` 选项。     |
