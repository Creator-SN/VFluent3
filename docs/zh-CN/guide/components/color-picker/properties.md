## 属性

---         
| 属性        | 类型            | 必填  | 默认值    | 说明              |
|:-----------:|:---------------:|:-----:|:---------:|:-----------------:|
| disabled    | boolean         | 否    | false     | disabled          |
| lang        | string          | 否    | "global"  | i18n              |
| model-value | [string]        | No    | undefined | v-model           |
| theme       | string          | 否    | 'global'  | theme             |
| type        | ['box', 'ring'] | No    | 'box'     | color-picker type |
