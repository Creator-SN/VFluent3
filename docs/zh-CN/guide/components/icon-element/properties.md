## 属性

---
| 属性         | 类型    | 必填  | 默认值           | 说明                             |
|:------------:|:-------:|:-----:|:----------------:|:--------------------------------:|
| value        | string  | 否    | 'AppIconDefault' | 参见组件的 `value` 选项。        |
| fontSize     | number  | 否    | 32               | 参见组件的 `fontSize` 选项。     |
| paddingWidth | number  | 否    | 2                | 参见组件的 `paddingWidth` 选项。 |
| iconButton   | boolean | 否    | false            | 参见组件的 `iconButton` 选项。   |
| borderRadius | number  | 否    | 6                | 参见组件的 `borderRadius` 选项。 |
| theme        | string  | 否    | 'global'         | 参见组件的 `theme` 选项。        |
| disabled     | boolean | 否    | false            | 参见组件的 `disabled` 选项。     |
| lang         | string  | 否    | "global"         | 参见组件的 `lang` 选项。         |
