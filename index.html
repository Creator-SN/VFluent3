Fluent Design on the Web

Components are shaped around Microsoft's Fluent Design language, with familiar Windows-style surfaces, depth, motion, Acrylic, and Reveal details.

Built for Vue 3

Designed for Vue 3 and Vite, with typed component entry points, plugin-friendly installation, and a workflow that feels natural in modern Vue projects.

Rich Component Coverage

From buttons, forms, menus, navigation, tables, and date pickers to panels, flyouts, shimmer, progress, and windowing controls, VFluent3 covers everyday product UI.

Themeable Experience

Light, dark, and base theme styles live with the component library, making it easier to keep interfaces consistent across different visual modes.

Fluent Icons Included

Segoe Fluent Icons and Fabric MDL2 icon fonts are bundled for quick access to the icon vocabulary used across Fluent-style interfaces.

Documentation That Ships Fast

VitePress-powered docs keep examples close to the components, so the library is easier to explore, maintain, and extend.
